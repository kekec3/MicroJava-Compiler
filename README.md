# 📘 MicroJava Compiler

A compiler for the **MicroJava** programming language, developed as part of a compiler construction project.

This project implements the full compilation pipeline—from lexical analysis to code generation—for a simplified Java-like language.

---

## 🚀 Features

* ✅ Lexical analysis (scanner)
* ✅ Syntax analysis (parser)
* ✅ Semantic analysis
* ✅ Code generation for MicroJava bytecode
* ✅ Error detection and reporting
* ✅ Example test programs included

---

## 🏗️ Project Structure

```
├── src/                # Source code
│   ├── lexer/         # Lexical analyzer
│   ├── parser/        # Syntax analyzer
│   ├── semantic/      # Semantic analysis
│   └── codegen/       # Code generation
├── test/              # Test programs (.mj files)
├── build/             # Compiled output
└── README.md
```

---

## ⚙️ Technologies Used

* Java
* JFlex (for lexical analysis)
* CUP (for parsing)

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/kekec3/MicroJava-Compiler.git
cd MicroJava-Compiler
```

### 2. Build the project

It's recommended to use the provided build.xml,

you can compile manually as well:

```bash
javac -d out src/**/*.java
```

### 3. Run the compiler

Cnahnge the file name inside MJCodeGeneratorTest.java and run it:

Trough the enviorment or terminal:

```bash
java test/MJCodeGeneratorTest.java
```
When you get program.obj file, just run it using build.xml or terminal:

```bash
start cmd /k java -cp lib\mj-runtime-1.1.jar rs.etf.pp1.mj.runtime.Run test/program.obj```
```

## 📚 About MicroJava

MicroJava is a simplified, educational programming language similar to Java, commonly used in compiler construction courses to demonstrate:

* Lexical and syntax analysis
* Semantic checks
* Intermediate code generation

---

## 🎯 Project Goals

* Understand how compilers work internally
* Implement each compilation phase from scratch
* Practice working with parsing tools like JFlex and CUP
